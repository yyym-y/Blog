乘法逆元

引入

在取模运算中，有这样几种操作：

(a±b)modp=(amodp±bmodp)modp(a×b)modp=(amodp×bmodp)modp

但加减乘除中除法并不遵循以上类似的规则，即：

abmodp≠(amodpbmodp)modp

既然除法没办法使用这种规律，那么我们想办法把除法转化为乘法

假设：

abmodp=a×binvmodp

那么我们现在就需要求出binv

具体的推导过程如下：

令ab≡xmodp , 则有 a⋅binv≡xmodp 。

两边同时乘上 b， 则有：

a≡x⋅bmodp 和 a⋅binv⋅b≡x⋅bmodp

之后根据模的减法得： a⋅(b⋅binv−1)≡0modp, 因为我们在正整数范围内讨论逆元，所以a≠0

所以： b⋅binv−1≡0modp , 即 ： b⋅binv≡1modp

所以我们得出了一下结论：

b⋅binv≡1modp

考虑一个问题：上述式子在任何情况下都能有binv 使式子成立吗：

我们不妨将上面得式子改写一下：

b⋅binv=k⋅p+1

再令g 为p , b 的最大公约数， 即：g=gcd(p,b) , 我们约定： g⋅b1=b , g⋅p1=p

则下列式子成立：

g⋅b1⋅binv=k⋅g⋅p1+1

我们可以将g 提出来：

g⋅(b1⋅binv−k⋅p1)=1

因为我们是在整数范围内考虑这些数， 所以可以得出这样两个结论：

  • g=1
  • b1⋅binv−k⋅p1 = 1

所以逆元存在的条件是： 当且仅当b 与 p 互质。

求乘法逆元

首先我们再次将逆元的结论抄下来：

b⋅binv≡1modp

那我们将如何求出这个binv 呢：

我们可以想到费马小定理

若 p 为质数， 则 ap−1≡1modp

所以我们可以将这个定理套到我们之前得出的结论里：

如果 p 为质数：

bp−1≡1modp

b⋅bp−2≡1modp == b⋅binv≡1modp

binv=bp−2

所以我们就获得了一个逆元：

如果p为质数:binv=bp−2

至于代码如何实现：我们可以使用快速幂来减少时间复杂度：

#include<bits/stdc++.h>
using namespace std;
typedef long long int lli;
lli P = 998244353;
lli mod(lli number) {return number % P;}
lli fast_pow(lli number, lli power){
    lli ans = 1LL; number %= P;
    while(power){
        if(power & 1) ans  = mod(ans * number);
        number = mod(number * number);
        power >>= 1;
    }
    return ans;
}
lli inv(lli number, lli p){
    return fast_pow(number, p - 2);
}
signed main()
{
    lli K; cin >> K;
    lli K_inv = inv(K, P);
    cout << ((K_inv * K) % P == 1);
}

 

费马小定理有一个弊端，那么就是要求 p 必须为质数：

为了可以适配更好的情况，我们可以使用拓展欧几里得定理， 它不要求p 为质数

拓展欧几里得定理 exgcd

exgcd 主要是求 a⋅x+b⋅y=gcd(a,b) 中的一组 x 和 y

那么就有以下推导过程：

因为 gcd(a,p)=1， 所以有：

a⋅x+p⋅y=gcd(a,p)=1

a⋅x=p⋅y+1

所以x=ainv

#include<bits/stdc++.h>
using namespace std;
typedef long long int lli;
lli P = 998244353;
lli mod(lli number) {return number % P;}
void exgcd(lli a, lli b, lli& x, lli& y){
    if(b == 0) {
        x = 1; y = 0;
        return;
    }
    exgcd(b, a % b, y, x);
    y -= (a / b) * x;
}
lli inv(lli number, lli p){
    lli x = 1, y = 0;
    exgcd(number, p, x, y);
    return (x % p + p) % p;
}
signed main()
{
    lli K; cin >> K;
    lli K_inv = inv(K, P);
    cout << ((K_inv * K) % P == 1);
}
